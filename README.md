## üíª Project Overview

**GoFox** is a service that simplifies the purchase of auto parts from international markets. The website acts as a landing page to present services, product catalog, and contact information.

### üåü Key Features
* **Responsive Design:** Fully adaptive layout that works seamlessly on desktops, tablets, and mobile devices (optimized via Media Queries).
* **Modern Layout Techniques:** Extensive use of **CSS Grid** for the product catalog and **Flexbox** for alignment.
* **Interactive Elements:** Hover effects on product cards, buttons, and social media icons.
* **Sections:** Hero Branding, About Us, Service Cards, Product Grid, Contact Form.
